## How Files Were Split

Large sections were split as follows to keep files under ~1.5 MB:

| Section | Strategy | Chunk Size |
|---|---|---|
| Posts (9.9 MB) | Chronological chunks | 1,000 posts per file |
| Shop Browsing (1.6 MB) | Chronological chunks | 2,500 items per file |
| Login History (1.9 MB) | Chronological chunks | 3,000 records per file |
| Watch History (1.9 MB) | Chronological chunks | 5,000 records per file |
| Followers / Following | Separated from Profile & Settings | one file each |
| All other sections | Kept as single files | — |

Multi-part files include a `_meta` object with: `part`, `of`, `items_in_part`, `total_items`, and `date_range` for easy navigation.
